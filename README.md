## Frenzy ISO

**The public files are a fraction of what's inside.**

The public repos show the surface — scripts, configs, a few utilities. Frenzy is the full build: a stripped and hardened Windows 11 ISO with registry edits and kernel drivers that aren't published anywhere, MSR and MMIO tuning wired into boot, and an integrated toolchain that makes it all work together.
